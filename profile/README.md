## 为什么静态住宅IP和其他IP差那么多？

先把概念理清楚，不然选起来容易绕晕。

市面上常见的IP分三类：

**机房IP（Datacenter IP）**：来自数据中心，速度快、价格便宜，但在各大平台眼里就是"明显的代理特征"。TikTok、Instagram、Amazon等平台的风控模型已经能大概率识别出来。适合低风险的数据采集，不适合账号运营。

**动态住宅IP（Rotating Residential IP）**：来自真实家庭网络，但地址会频繁变动。适合批量抓取数据，不适合需要长期保持同一身份的账号（账号系统会认为你"换了设备"）。

## 怎么看一个静态住宅IP够不够"真"？

最后给个实用的自查方式，不管买谁家都可以用：

**1. 查ASN归属**
IP归属的AS号应该是真实ISP运营商（如Comcast、AT&T、T-Mobile、Spectrum等），而不是数据中心或CDN提供商。用ipinfo.io直接查。

**2. 看IP在数据库里的类型标注**
MaxMind GeoIP2、IP2Location都有"residential"和"datacenter"的类型区分，真实住宅IP会被标记为residential。

**3. 黑名单状态**
用MXToolbox或类似工具查询IP是否在主要RBL黑名单里。新号段、独享IP被标记的可能性更低。

**4. 实际跑目标平台**
说到底，最直接的验证方式就是拿着IP去目标平台（TikTok、Instagram、亚马逊）跑一跑，看推流数据、账号权重是否正常。
